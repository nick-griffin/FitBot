# FitBot
AI-powered fitness and nutrition coaching app

Android fitness application using Kotlin and Jetpack Compose to generate personalized workout and nutrition plans



FastAPI backend to handle user data and integrate AI-generated responses through external API calls




Room database to store and manage user profile and fitness data
